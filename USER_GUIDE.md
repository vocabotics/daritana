## Project Management

### Creating a New Project

1. **Navigate to Projects**
   - Click "Projects" in sidebar
   - Or press `Ctrl+P`

2. **Click "New Project"**
   - Or press `Ctrl+N`

3. **Fill Project Details**
   - **Project Name**: Descriptive name
   - **Project Type**: Select from:
     - Residential Design
     - Commercial Design
     - Interior Design
     - Renovation
     - Industrial
     - Mixed-Use Development
   - **Client Information**:
     - Client name
     - Contact details
     - Company (if applicable)
   - **Location**:
     - Site address
     - City and state
     - GPS coordinates (optional)
   - **Timeline**:
     - Start date
     - Expected completion date
     - Key milestones
   - **Budget**:
     - Total budget (RM)
     - Budget breakdown by phase
   - **Description**:
     - Project scope
     - Special requirements
     - Client expectations

4. **Assign Team Members**
   - Select Project Lead
   - Add Architects
   - Add Engineers
   - Add Contractors
   - Add other team members

5. **Set Project Status**
   - Planning
   - In Progress
   - On Hold
   - Completed
   - Archived

6. **Click "Create Project"**

## Task Management

### Kanban Board Overview

The Kanban board visualizes your workflow with four columns:

1. **To Do**: New tasks and backlog
2. **In Progress**: Active work
3. **Review**: Awaiting approval
4. **Done**: Completed tasks

### Creating Tasks

1. **Click "New Task"** or press `Ctrl+N`

2. **Fill Task Details**:
   - **Title**: Clear, descriptive name
   - **Description**: Detailed task information
   - **Project**: Select project
   - **Assigned To**: Team member
   - **Priority**:
     - üî¥ Urgent
     - üü† High
     - üü° Medium
     - üü¢ Low
   - **Due Date**: Deadline
   - **Estimated Time**: Hours/days
   - **Category**: Design, Engineering, Client, etc.
   - **Tags**: Custom labels
   - **Attachments**: Related files

3. **Click "Create Task"**

### Managing Tasks

**Moving Tasks:**
- Drag and drop between columns
- Task status updates automatically
- Team members get notified

**Task Actions:**
- **View Details**: Click task card
- **Edit**: Modify task information
- **Comment**: Add notes and updates
- **Attach Files**: Upload relevant documents
- **Set Dependencies**: Link related tasks
- **Clone**: Create similar task
- **Delete**: Remove task

### Task Filters

Filter tasks to focus on specific work:

- **By Project**: Show only specific project tasks
- **By Assignee**: Your tasks, team member tasks, unassigned
- **By Priority**: Urgent, high, medium, low
- **By Due Date**: Overdue, today, this week, this month
- **By Category**: Design, engineering, client, etc.

### Task Views

1. **Board View**: Visual Kanban board
2. **List View**: Detailed table
3. **Calendar View**: Tasks on calendar
4. **Timeline View**: Gantt-style timeline

## Document Management

### Document Categories

Organize documents by category:

- **Architectural**: Floor plans, elevations, sections, details
- **Structural**: Structural drawings and calculations
- **MEP**: Mechanical, electrical, plumbing systems
- **Site Plans**: Site layouts, surveys, civil plans
- **Specifications**: Material specs, technical standards
- **Reports**: Site reports, inspection reports
- **Photos**: Site photos, progress photos, references
- **Contracts**: Agreements, change orders

### Uploading Documents

1. **Navigate to Documents Page**

2. **Click "Upload"** or drag & drop files

3. **Select Files**:
   - Single file
   - Multiple files
   - Folder upload

4. **Add Metadata**:
   - Category
   - Project
   - Document type
   - Version number
   - Description
   - Tags

5. **Click "Upload"**

### Document Version Control

**How Version Control Works:**

1. **Upload New Version**
   - Select existing document
   - Click "Upload New Version"
   - Upload updated file
   - Add version notes

2. **View Version History**
   - Click document
   - Select "Version History"
   - See all versions with timestamps
   - Compare versions side-by-side

3. **Restore Previous Version**
   - Open version history
   - Select version to restore
   - Click "Restore"
   - Becomes current version (old version preserved)

### Document Review Workflows

**Starting a Review Session:**

1. Click "Start Review" on document
2. Select reviewers
3. Add review instructions
4. Set review deadline
5. Click "Start"

**Reviewers Will:**
- Receive notification
- Access document with markup tools
- Add comments and annotations
- Approve or request changes

**Markup Tools:**

2D Documents:
- Pen tool for freehand drawing
- Highlighter for marking areas
- Text comments
- Stamps (Approved, Rejected, etc.)
- Measurement tools

3D Models:
- Navigate 3D model
- Section cuts
- Measurement tools
- 3D annotations
- Walkthrough mode

### Document Sharing

**Share Documents:**

1. Select document(s)
2. Click "Share"
3. Choose method:
   - Share link
   - Email recipients
   - Set password (optional)
   - Set expiration date
4. Click "Generate Share Link"

**Permission Levels:**
- **View Only**: Can view but not download
- **Download**: Can view and download
- **Comment**: Can view, download, and comment
- **Edit**: Full editing permissions

## Tips and Best Practices

### Project Management Tips

1. **Keep Project Information Updated**
   - Update status regularly
   - Log all important decisions
   - Document changes immediately

2. **Use Milestones Effectively**
   - Set clear, measurable milestones
   - Link tasks to milestones
   - Review milestone progress weekly

3. **Organize with Tags**
   - Create consistent tag naming
   - Use tags for filtering
   - Color-code by importance

4. **Regular Team Updates**
   - Daily standups for active projects
   - Weekly review meetings
   - Monthly retrospectives

### Task Management Tips

1. **Write Clear Task Descriptions**
   - Use action verbs
   - Include acceptance criteria
   - Add relevant context

2. **Prioritize Effectively**
   - Use priority levels consistently
   - Review priorities daily
   - Focus on high-priority first

3. **Break Down Large Tasks**
   - Limit task size to 1-2 days
   - Create subtasks for big items
   - Link related tasks

4. **Set Realistic Deadlines**
   - Add buffer for unknowns
   - Consider dependencies
   - Account for team availability

### Document Management Tips

1. **Use Consistent Naming**
   - Include date in filename
   - Use version numbers
   - Describe document clearly

2. **Organize by Category**
   - Use all available categories
   - Keep consistent structure
   - Archive old documents

3. **Add Metadata**
   - Fill in all fields
   - Use descriptive tags
   - Add searchable keywords

4. **Version Control Best Practices**
   - Upload new versions promptly
   - Add meaningful version notes
   - Keep important versions

### Collaboration Tips

1. **Set Clear Status**
   - Update status when changing activities
   - Use custom status for clarity
   - Set realistic availability

2. **Effective Communication**
   - @Mention for urgent items
   - Use appropriate channels
   - Keep messages professional
   - Respond promptly

3. **Share Context**
   - Provide background info
   - Link relevant documents
   - Explain technical terms

4. **Virtual Meetings**
   - Join on time
   - Mute when not speaking
   - Use video when possible
   - Share screen for clarity

## Troubleshooting

### Common Issues

**Can't Log In:**
1. Check internet connection
2. Verify email and password
3. Clear browser cache
4. Try password reset
5. Contact administrator

**Page Not Loading:**
1. Refresh page (F5)
2. Clear browser cache
3. Check internet connection
4. Try different browser
5. Check system status

**Files Not Uploading:**
1. Check file size (<100MB)
2. Verify file type allowed
3. Check internet connection
4. Try different browser
5. Contact support

**Notifications Not Working:**
1. Check notification settings
2. Verify email address
3. Check spam folder
4. Enable browser notifications
5. Update notification preferences

**Can't See Project:**
1. Verify project access
2. Check filters applied
3. Verify not archived
4. Contact project lead
5. Check permissions
